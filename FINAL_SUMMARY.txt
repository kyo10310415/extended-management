✨ 目的:
   スプレッドシートベースの延長管理システムをWebアプリ化し、
   視認性と操作性を大幅に向上させる

🎯 主要機能:
   ✓ ダッシュボード - 統計情報の一目確認
   ✓ ヒアリング一覧 - 4ヶ月目生徒の自動表示・管理
   ✓ 延長審査一覧 - 5ヶ月目生徒の自動表示・管理
   ✓ 生徒情報マスタ - 全生徒データの一覧・検索

   ⚛️  React 18 - UIフレームワーク
   🎨 TailwindCSS - スタイリング
   ⚡ Vite - ビルドツール

Backend:
   🟢 Node.js + Express - サーバー
   📝 Notion API - 生徒基本情報
   📊 Google Sheets API - フォーム更新日
   🗄️  PostgreSQL - 手動入力データ

Deployment:
   🚀 Render - Web Service + Database
   🐙 GitHub - バージョン管理
